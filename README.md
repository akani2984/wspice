# wspice
spice client for desktop and mobile, WITHOUT any pro versions and ads, based on spice-html5
# usage 
on your spice server: <br>
websockify  --web=place-to-wspice adddress-for-listening  spice-server-spice <br>
for example , <br>
websockify  --web=/Data/Work/wspice 0.0.0.0:9002 127.0.0.1:5901 <br>
then access http://you-server-ip:9002  to use the client. <br>
use you-server-ip:9002 to connect the spice server
